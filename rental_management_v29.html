🔥 مزامنة البيانات — Firebase غير متصل
✅
متصل بـ Firebase — البيانات تتزامن تلقائياً
آخر مزامنة: —
⚡ خطوات الإعداد:
① اذهب إلى firebase.google.com وسجّل
② أنشئ مشروعاً → Realtime Database → Start in test mode
③ انسخ رابط قاعدة البيانات (Database URL)
④ من ⚙️ Project Settings انسخ API Key و Project ID
✈️ بوت تليغرام — إرسال تلقائي غير مُضبوط
⚡ خطوات الإعداد (١٠ دقائق فقط)
① افتح تليغرام وابحث عن @BotFather
② أرسل له: /newbot واختر اسماً للبوت
③ ستحصل على Bot Token — انسخه وضعه أدناه
④ افتح محادثة مع البوت واضغط Start ثم اضغط "احصل على Chat ID" أدناه
⑤ أضف أرقام Chat IDs للمستلمين ثم اضغط اختبار
احصل عليه من @BotFather
أضف الـ Chat ID لكل شخص تريد إرسال الإشعارات له
🔗 فتح BotFather
⚡ ربط EmailJS — إرسال البريد غير مُضبوط
خطوات الإعداد:
① سجّل في emailjs.com (مجاني ٢٠٠ بريد/شهر)
② أضف Email Service (Gmail/Outlook) وأنشئ Email Template
③ في القالب: To = {{to_email}} — HTML Body = {{{html_body}}}
④ انسخ المفاتيح من Account → API Keys
🔗 فتح لوحة EmailJS
